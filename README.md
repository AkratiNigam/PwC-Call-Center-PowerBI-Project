# PwC Call Center PowerBI Project
The objective of this project was to analyze customer service data from a call center and create interactive dashboards to help stakeholders understand performance trends, customer satisfaction, and operational efficiency.
## Key Features
1. Built dynamic and interactive Power BI dashboards
2. Analyzed key performance indicators (KPIs) such as:
   Call resolution rates,
   Average handling time,
   Customer satisfaction scores,
   Agent performance metrics
3. Identified trends and patterns to support data-driven decision-making
4. Used data visualization techniques to present insights clearly and effectively
## Tools & Technologies
1. Power BI
2. Data Cleaning & Transformation (Power Query)
3. Data Modeling
4. DAX (Data Analysis Expressions)
## Insights Delivered
1. Highlighted areas of improvement in call resolution efficiency
2. Identified top-performing and underperforming agents
3. Revealed patterns affecting customer satisfaction
4. Provided recommendations to enhance overall service quality
## Outcome
This project demonstrates my ability to transform raw data into meaningful insights and build dashboards that support business decisions in a real-world scenario.
