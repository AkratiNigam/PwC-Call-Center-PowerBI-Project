# PwC Call Center PowerBI Project
The objective of this project was to analyze customer service data from a call center and create interactive dashboards to help stakeholders understand performance trends, customer satisfaction, and operational efficiency.
## Key Features
-Built dynamic and interactive Power BI dashboards
-Analyzed key performance indicators (KPIs) such as:
-Call resolution rates
-Average handling time
-Customer satisfaction scores
-Agent performance metrics
-Identified trends and patterns to support data-driven decision-making
-Used data visualization techniques to present insights clearly and effectively
## Tools & Technologies
-Power BI
-Data Cleaning & Transformation (Power Query)
-Data Modeling
-DAX (Data Analysis Expressions)
## Insights Delivered
Highlighted areas of improvement in call resolution efficiency
-Identified top-performing and underperforming agents
-Revealed patterns affecting customer satisfaction
-Provided recommendations to enhance overall service quality
## Outcome
This project demonstrates my ability to transform raw data into meaningful insights and build dashboards that support business decisions in a real-world scenario.
